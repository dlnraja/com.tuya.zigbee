Easiest method - Just enter your Smart Life / Tuya Smart email and password.
Requires IoT credentials saved in App Settings (one-time setup).
First time? Save credentials in App Settings
  1. Go to iot.tuya.com and create a free account
  2. Cloud > Create Cloud Project > select Smart Home
  3. Subscribe to IoT Core + Authorization APIs
  4. Devices > Link App Account > scan QR with Smart Life app
  5. Copy Access ID + Access Secret from Overview tab
  6. Go to this app's Settings page and save them there
After this one-time setup, you only need email + password to pair devices.
