# Universal Tuya Zigbee App for Homey

This app adds support for Tuya Zigbee and WiFi devices to Homey.

## Features
- Support for 260+ drivers and 3000+ device fingerprints.
- High-stability Nexus Engine architecture.
- SDK 3 compliant.
- Community-driven device enrichment.
