# Driver light-tuya-tuya-floor-269

## Description
Driver pour appareil light tuya tuya-floor-269

## Capabilities
- onoff (par défaut)

## Support
- Test mode only
- Non publié sur Homey Store

## Installation
1. Copier ce dossier dans `drivers/tuya/light/tuya/tuya-floor-269/`
2. Redémarrer l'app Homey
3. L'appareil sera automatiquement détecté

## Configuration
Modifier `driver.compose.json` pour ajuster les capabilities et paramètres Zigbee.
