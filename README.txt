DEVICE PAIRING

1. Open Homey app
2. Go to Devices → Add Device
3. Select "Universal Tuya Zigbee"
4. Choose device type by FUNCTION (not brand)
5. Put device in pairing mode (usually press/hold button)
6. Homey will detect and configure device automatically

PAIRING TIPS:
- Keep device close to Homey during pairing (< 2 meters)
- Check device manual for specific pairing procedure
- Some devices require reset before pairing
- If pairing fails, try moving closer to Homey
- Check Homey's Zigbee network is active

TROUBLESHOOTING

DEVICE NOT PAIRING:
1. Reset device (check manual for procedure)
2. Move closer to Homey (< 2 meters)
3. Ensure Zigbee network is active
4. Check device is in pairing mode
5. Try removing batteries and reinserting (battery devices)

DEVICE NOT RESPONDING:
1. Check Zigbee network status in Homey
2. Check device battery (battery-powered devices)
3. Move device closer or add Zigbee router
4. Repair device through Homey app

WRONG DEVICE TYPE SELECTED:
1. Remove device from Homey
2. Re-add with correct function-based driver
3. Device will be reconfigured automatically
