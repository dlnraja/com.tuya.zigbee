// Driver light-tuya-tuya-table-5

#// Description
Driver pour appareil light tuya tuya-table-5

#// Capabilities
- onoff (par défaut)

#// Support
- Test mode only
- Non publié sur Homey Store

#// Installation
1. Copier ce dossier dans `drivers/tuya/light/tuya/tuya-table-5/`
2. Redémarrer l'app Homey
3. L'appareil sera automatiquement détecté

#// Configuration
Modifier `driver.compose.json` pour ajuster les capabilities et paramètres Zigbee.
