// Driver siren-ikea-ikea-buzzer-232

#// Description
Driver pour appareil siren ikea ikea-buzzer-232

#// Capabilities
- onoff (par défaut)

#// Support
- Test mode only
- Non publié sur Homey Store

#// Installation
1. Copier ce dossier dans `drivers/zigbee/siren/ikea/ikea-buzzer-232/`
2. Redémarrer l'app Homey
3. L'appareil sera automatiquement détecté

#// Configuration
Modifier `driver.compose.json` pour ajuster les capabilities et paramètres Zigbee.
