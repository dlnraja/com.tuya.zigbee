## 🔍 PROBLÈME: "Zigbee Inconnu"

C'est **normal et attendu** pour les devices appairés avec v4.9.49 (version cassée):

**Pourquoi**:
- v4.9.49 configurait incorrectement les clusters Zigbee
- Les devices ont été "mal appairés" avec mauvaises métadonnées
- v4.9.50 ne reconnaît pas ces anciennes configurations incorrectes

**Solution**: Re-pairing complet 🔄

## ✅ PROCÉDURE DE RE-PAIRING

### Pour CHAQUE device affecté:

**1. RETIRER le device de Homey**:
- Ouvrir app Homey mobile
- Aller dans Devices
- Trouver le device "Zigbee Inconnu"
- Appuyer sur ⚙️ (Settings)
- Scroller en bas → "Remove device"
- Confirmer

**2. FACTORY RESET le device physique**:

**USB Outlet (dongle routeur)**:
- Débrancher de l'alimentation
- Maintenir le bouton physique enfoncé
- Rebrancher (tout en maintenant)
- LED clignote rapidement = Reset OK
- Relâcher après 10 secondes

**Climate Sensor (capteur température/humidité)**:
- Ouvrir le compartiment batterie
- Retirer la batterie 10 secondes
- Maintenir le bouton reset (petit trou)
- Remettre la batterie (en maintenant)
- LED clignote = Reset OK

**3. RE-PAIRING avec v4.9.50**:
- App Homey → Devices → "+" (Add device)
- Chercher "Universal Tuya Zigbee"
- Sélectionner le type:
  * USB Outlet → "USB Outlet" (2-port ou autre)
  * Climate → "Climate Sensor" ou "Temperature Sensor"
- Distance <2m du Homey
- Attendre détection (30-60 secondes)
- Nommer le device
- ✅ Terminé!

**4. VÉRIFIER que ça marche**:
- Ouvrir le device dans Homey
- Vérifier capabilities visibles:
  * USB: On/Off pour chaque port
  * Climate: Température, Humidité, Batterie
- Tester On/Off (pour USB)
- Attendre 5 minutes pour premières données

## 📊 POURQUOI C'EST NÉCESSAIRE?

v4.9.49 → v4.9.50 a changé la **structure fondamentale** de configuration:

```javascript
// v4.9.49 (CASSÉ):
cluster: 1  // ID numérique brut

// v4.9.50 (CORRECT):
cluster: CLUSTER.POWER_CONFIGURATION  // Objet complet
```

Les devices appairés avec v4.9.49 ont des métadonnées **incompatibles** avec v4.9.50. Le re-pairing crée des métadonnées **propres et correctes**.
