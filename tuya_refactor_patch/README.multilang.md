# Tuya Zigbee for Homey

Application officielle pour les appareils Zigbee Tuya sur Homey.

## Fonctionnalités
- Support complet du SDK Homey v3
- Compatibilité avec des centaines d'appareils Tuya
- Mise à jour mensuelle automatique des métadonnées
- Documentation multilingue

## Installation
```bash
npm install
npx homey app run
```
