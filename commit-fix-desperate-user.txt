RÈGLE FONDAMENTALE SDK3:

❌ JAMAIS faire setup*() AVANT super.onNodeInit()
✅ TOUJOURS faire setup*() APRÈS super.onNodeInit()

Raison: 
- super.onNodeInit() crée zclNode
- zclNode.endpoints n'existe pas avant
- registerCapability() requiert zclNode
- clusters ne sont pas disponibles avant init

ORDRE CORRECT:
```javascript
async onNodeInit() {
  // 1. Log init start
  this.log('Device initializing...');
  
  // 2. Initialize base FIRST
  await super.onNodeInit();
  
  // 3. THEN setup everything else
  await this.setupMultiEndpoint();
  await this.setupIASZone();
  await this.setupAttributeReporting();
  
  // 4. Log completion
  this.log('Device ready');
}
```
