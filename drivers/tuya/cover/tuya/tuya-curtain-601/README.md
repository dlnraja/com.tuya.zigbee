# Driver cover-tuya-tuya-curtain-601

## Description
Driver pour appareil cover tuya tuya-curtain-601

## Capabilities
- onoff (par défaut)

## Support
- Test mode only
- Non publié sur Homey Store

## Installation
1. Copier ce dossier dans `drivers/tuya/cover/tuya/tuya-curtain-601/`
2. Redémarrer l'app Homey
3. L'appareil sera automatiquement détecté

## Configuration
Modifier `driver.compose.json` pour ajuster les capabilities et paramètres Zigbee.
