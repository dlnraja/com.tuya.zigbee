# Driver siren-aqara-aqara-alarm-922

## Description
Driver pour appareil siren aqara aqara-alarm-922

## Capabilities
- onoff (par défaut)

## Support
- Test mode only
- Non publié sur Homey Store

## Installation
1. Copier ce dossier dans `drivers/zigbee/siren/aqara/aqara-alarm-922/`
2. Redémarrer l'app Homey
3. L'appareil sera automatiquement détecté

## Configuration
Modifier `driver.compose.json` pour ajuster les capabilities et paramètres Zigbee.
