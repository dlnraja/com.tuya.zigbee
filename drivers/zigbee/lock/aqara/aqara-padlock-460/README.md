# Driver lock-aqara-aqara-padlock-460

## Description
Driver pour appareil lock aqara aqara-padlock-460

## Capabilities
- onoff (par défaut)

## Support
- Test mode only
- Non publié sur Homey Store

## Installation
1. Copier ce dossier dans `drivers/zigbee/lock/aqara/aqara-padlock-460/`
2. Redémarrer l'app Homey
3. L'appareil sera automatiquement détecté

## Configuration
Modifier `driver.compose.json` pour ajuster les capabilities et paramètres Zigbee.
