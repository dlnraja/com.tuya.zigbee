# Driver plug-ikea-ikea-smoke-94

## Description
Driver pour appareil plug ikea ikea-smoke-94

## Capabilities
- onoff (par défaut)

## Support
- Test mode only
- Non publié sur Homey Store

## Installation
1. Copier ce dossier dans `drivers/zigbee/plug/ikea/ikea-smoke-94/`
2. Redémarrer l'app Homey
3. L'appareil sera automatiquement détecté

## Configuration
Modifier `driver.compose.json` pour ajuster les capabilities et paramètres Zigbee.
