# Driver cover-ikea-ikea-garage-65

## Description
Driver pour appareil cover ikea ikea-garage-65

## Capabilities
- onoff (par défaut)

## Support
- Test mode only
- Non publié sur Homey Store

## Installation
1. Copier ce dossier dans `drivers/zigbee/cover/ikea/ikea-garage-65/`
2. Redémarrer l'app Homey
3. L'appareil sera automatiquement détecté

## Configuration
Modifier `driver.compose.json` pour ajuster les capabilities et paramètres Zigbee.
