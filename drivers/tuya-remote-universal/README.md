# Télécommande Tuya universelle pour scènes

## Description
Driver universel pour les devices Tuya Zigbee de type remote.

## Capacités
- `alarm_battery`

## Installation
1. Installer l'application Tuya Zigbee (Lite)
2. Ajouter le device via l'interface Homey
3. Suivre les instructions de pairing
