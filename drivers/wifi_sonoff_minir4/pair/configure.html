eWeLink / SONOFF WiFi Setup

100% local LAN control - no cloud needed after setup

How to get Device ID and API Key:
  1. Open eWeLink app > Device > Settings > Device Info > Device ID
  2. For API Key: use eWeLink Web or a tool like SonoffLAN
  3. For DIY mode devices: API Key is not needed (leave empty)
  4. IP: find in your router's DHCP table or use mDNS scan
Required for encrypted LAN mode, not needed for DIY mode
