CRITICAL: Fix BaseHybridDevice battery monitoring - Replace deprecated registerCapability with SDK3 cluster listeners (v4.9.20)

ROOT CAUSE:
BaseHybridDevice.js used deprecated SDK2 registerCapability() method causing
'expected_cluster_id_number' TypeError in ALL battery-powered devices.

DIAGNOSTIC LOG: 8e9f30c9-a118-4e9a-ac97-42a876f6c092
User reported: "Issue encore pas de data sur tout les drivers visibles"

FIXES APPLIED:
1. Replaced registerCapability with SDK3 direct cluster listeners
2. endpoint.clusters.genPowerCfg.on('attr.batteryPercentageRemaining', ...)
3. configureAttributeReporting with numeric cluster ID (1)
4. Temporarily disabled AC/DC monitoring (needs separate SDK3 refactor)

AFFECTS:
- ALL battery-powered devices
- Presence sensors
- Climate sensors
- Emergency SOS buttons
- Wireless buttons (3/4 gang)
- Any device extending BaseHybridDevice

Battery data will now appear correctly for all affected devices.
