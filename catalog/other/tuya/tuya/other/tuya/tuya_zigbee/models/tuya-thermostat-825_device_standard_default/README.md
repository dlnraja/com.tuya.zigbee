// Driver plug-tuya-tuya-thermostat-825

#// Description
Driver pour appareil plug tuya tuya-thermostat-825

#// Capabilities
- onoff (par défaut)

#// Support
- Test mode only
- Non publié sur Homey Store

#// Installation
1. Copier ce dossier dans `drivers/tuya/plug/tuya/tuya-thermostat-825/`
2. Redémarrer l'app Homey
3. L'appareil sera automatiquement détecté

#// Configuration
Modifier `driver.compose.json` pour ajuster les capabilities et paramètres Zigbee.
