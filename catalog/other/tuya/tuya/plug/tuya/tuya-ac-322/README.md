# Driver plug-tuya-tuya-ac-322

## Description
Driver pour appareil plug tuya tuya-ac-322

## Capabilities
- onoff (par défaut)

## Support
- Test mode only
- Non publié sur Homey Store

## Installation
1. Copier ce dossier dans `drivers/tuya/plug/tuya/tuya-ac-322/`
2. Redémarrer l'app Homey
3. L'appareil sera automatiquement détecté

## Configuration
Modifier `driver.compose.json` pour ajuster les capabilities et paramètres Zigbee.
