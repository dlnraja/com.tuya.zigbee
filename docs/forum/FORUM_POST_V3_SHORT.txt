🚀 UNIVERSAL TUYA ZIGBEE v3.0.0 - MAJOR RELEASE

Hi everyone! 👋

I'm excited to announce **version 3.0.0** of Universal Tuya Zigbee - a major milestone representing a complete architectural evolution!

━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 WHAT'S NEW

1️⃣ **Tuya DP Engine** - Revolutionary Architecture
→ Centralized Data Point handling
→ New devices = simple JSON config (no code!)
→ 90% code reduction potential
→ Scalable to 500+ devices

2️⃣ **Local-First Philosophy** - Clearly Documented
→ 10-50ms latency (vs 500-2000ms cloud)
→ Works 100% offline
→ Total privacy guaranteed
→ 40+ pages comprehensive guide

3️⃣ **Complete Transparency** - Verifiable Claims
→ CI/CD validates every change
→ Device matrix auto-generated
→ All build artifacts public
→ No "trust us" - now "verify yourself"

4️⃣ **Professional Documentation** - 115+ pages
→ Technical guides
→ Comparison with alternatives
→ Coverage methodology explained
→ Device request templates

🤝 OUR POSITIONING

This app focuses on **local Zigbee control** for Tuya devices.
Other apps (like Tuya Cloud) are great for WiFi devices via cloud.

→ Both approaches are valuable!
→ Choose based on your needs
→ We're complementary, not competitive

📖 [Read detailed comparison](https://github.com/dlnraja/com.tuya.zigbee/blob/master/docs/WHY_THIS_APP.md)

🚀 INSTALLATION

Via Homey CLI:
```
npm install -g homey
git clone https://github.com/dlnraja/com.tuya.zigbee
cd com.tuya.zigbee
npm install
homey app run
```

Homey App Store: Coming after stability testing!
