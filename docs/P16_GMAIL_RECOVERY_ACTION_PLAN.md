2. Sign in with `senetmarne@gmail.com`

4. Click "Generate" — you'll get a 16-character password (xxxx xxxx xxxx xxxx)
5. Copy it
6. Go to https://github.com/dlnraja/com.tuya.zigbee/settings/secrets/actions
7. Click on `GMAIL_APP_PASSWORD` → "Update"
8. Paste the new password (no spaces, e.g., `xxxxxxxxxxxxxxxx`)
9. Click "Update secret"

### Step 2: Refresh `GMAIL_REFRESH_TOKEN`

1. Go to https://developers.google.com/oauthplayground/
2. Click the gear icon ⚙️ → "OAuth 2.0 configuration"
3. Check "Use your own OAuth credentials"
4. Enter:
   - OAuth Client ID: `GMAIL_CLIENT_ID` (from your existing secret)
   - OAuth Client Secret: `GMAIL_CLIENT_SECRET` (from your existing secret)
5. In "Step 1: Select & authorize APIs", find "Gmail API v1" → select `https://mail.google.com/`
6. Click "Authorize APIs"
7. Sign in with `senetmarne@gmail.com` if prompted
8. Click "Allow"
9. In "Step 2: Exchange authorization code for tokens", click "Exchange authorization code for tokens"
10. Copy the `refresh_token` value
11. Go to https://github.com/dlnraja/com.tuya.zigbee/settings/secrets/actions
12. Click on `GMAIL_REFRESH_TOKEN` → "Update"
13. Paste the new token
14. Click "Update secret"
