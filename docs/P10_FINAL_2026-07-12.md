## Explicit Exception Pattern

**Quand** : Pour supprimer **ses propres commentaires** sur le repo upstream (Dylan est l'auteur).

**Conditions** (toutes requises) :
1. `ALLOW_UPSTREAM_WRITE=true` env var (acknowledgment explicite)
2. `--confirm` CLI flag (pas d'accident)
3. `GH_TOKEN` env var (auth + scope proof)

**Outil** : `tools/ci/delete-johan-comments.js` (Node.js, portable)

**Modes** :
- **Default (no flags)** : REVIEW ONLY — affiche les URLs + extraits, zéro appel API
- **Avec 3 confirmations** : ACTUAL DELETE — 5s délai pour Ctrl+C avant de commencer

**Sécurité additionnelle** :
- 5 secondes d'attente avant le premier DELETE (Ctrl+C possible)
- Rate limit 250ms entre chaque DELETE (4 req/s)
- 404 → "Already gone" (pas d'erreur)
- 403/401 → loggé mais continue
- `--max=N` pour tester
- `ISSUE_FILTER=N` pour une seule issue
- Report JSON avec status par commentaire
