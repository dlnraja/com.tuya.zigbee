Pour publier l'application, vous devez exécuter cette commande dans le terminal:

homey app publish

Puis répondre:
1. Y (Yes) pour mettre à jour la version
2. Choisir le type de version: patch, minor, ou major
3. Confirmer la publication

Alternative - Publication via GitHub Release:
1. Créer une release sur GitHub
2. Tagger avec v4.9.273 (ou la nouvelle version)
3. Le workflow GitHub Actions publiera automatiquement

Commande complète:
cd "C:\Users\HP\Desktop\homey app\tuya_repair"
homey app publish

Note: Vous devez être connecté avec 'homey login' et avoir le HOMEY_TOKEN configuré.
