# KNOWN_ISSUES.md - Known Issues & Limitations

> **Last Updated**: 2026-06-15 | **Version**: 9.0.36

## Firmware-Level Issues (Cannot Fix in App)

| Issue | Affected Devices | Symptom | Workaround |
|-------|-----------------|---------|------------|
| **TS0601 Time Sync** | LCD climate sensors (_TZE284_*) | Device ignores Zigbee time responses | App retries periodically; some data points may show stale timestamps |
| **MCU Time Format Mismatch** | TRVs, LCD sensors, weather stations | MCU expects different time payload format | App supports 23 formats with auto-detection via guessFormat(); fallback chain tries alternatives |
| **MCU UART v3.3+ Sequence Echo** | TZE200/TZE204/TZE284 MCU devices | MCU requires sequence number echo in time response | TUYA_SEQ_10 format echoes sequence from device request |
| **ZT08 Clock Drift** | ZT08 weather stations (_TZE200_rhgsbacq) | Clock drifts after time sync without commit | DP17 commit trigger (write false) sent after time sync |
| **Battery 0%** | TS0044 buttons (_TZ3000_wkai4ga5) | Reports 0% always regardless of actual charge | Firmware bug; voltage-based fallback may help if voltage DP is available |
| **Cloud-only MCU** | Some TS0601 variants | MCU ignores all local Zigbee commands | No local workaround; device requires cloud connection |
| **Curtain position invert** | Quoya M515EGBZTN, some cover motors | 0=open, 100=closed (inverted from standard 0=closed, 100=open) | Use `positionInvert: true` in cover profile |
| **IAS Zone false alarms** | Some Tuya motion sensors | Spurious zone status changes on network rejoin | 1.5s deduplication window filters most false triggers |

## Historical Issues (Fixed)

| Issue | Version Fixed | Root Cause | Fix Applied |
|-------|--------------|------------|-------------|
| **Double-division bug** | v5.11.15 | AdaptiveDataParser + dpMappings both dividing values | TuyaEF00Manager skips auto-convert when dpMappings divisor !== 1 |
| **OOM crash (fingerprints)** | v8.5.7-v9.0.0 | `data/fingerprints.json` (11.5MB) parsed via UTF-16 string at startup | Buffer-based direct JSON parsing + lazy loading |
| **AggregateError** | v8.5.34 | Empty `manufacturerName: []` in 182 synthetic hybrid drivers | Dual-layer pairing architecture enforced; all empty arrays eliminated |
| **Phantom capabilities** | v8.0.0 | Fallback to `HOBEIAN_10G_MULTI` for unknown model IDs | Switched to `HOBEIAN_ZG204ZM_FALLBACK` which has minimal capabilities |
| **Double-parsing** | v8.2.0 | ZCL cluster listeners conflict with Tuya DP handlers | Check `!this.clusterHandlers?.temperatureMeasurement` before setup |
| **getTriggerCard deprecated** | v7.5.26 | 45 drivers using deprecated Homey SDK3 API | Migrated to `getDeviceTriggerCard()` |
| **OAuth white screen** | v7.5.26 | Architectural constraint with Homey OAuth flow | Closed -- not fixable at app level |
