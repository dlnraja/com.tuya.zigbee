# Flow Card Argument Types Reference
> v9.0.40 | Complete list of supported Homey SDK3 flow card argument types

## Supported Types

| Type | Description | JSON Example | Use Case |
|------|-------------|-------------|----------|
| `string` | Free text input | `{ "type": "string", "name": "label", "title": {"en": "Label"} }` | Device names, custom text |
| `number` | Numeric input | `{ "type": "number", "name": "value", "title": {"en": "Value"} }` | Counts, thresholds |
| `boolean` | True/false toggle | `{ "type": "boolean", "name": "enabled", "title": {"en": "Enabled"} }` | On/off states |
| `range` | Numeric slider | `{ "type": "range", "name": "brightness", "min": 0, "max": 100, "step": 1 }` | Brightness, temperature |
| `dropdown` | Dropdown select | `{ "type": "dropdown", "name": "mode", "values": [{"id":"auto","label":"Auto"},{"id":"manual","label":"Manual"}] }` | Mode selection |
| `device` | Device picker | `{ "type": "device", "name": "target", "filter": "capabilities=onoff" }` | Cross-device triggers |
| `color` | Color picker | `{ "type": "color", "name": "color", "title": {"en": "Color"} }` | Light color |
| `image` | Image (trigger token) | `{ "type": "image", "name": "snapshot" }` | Camera snapshots |
| `autocomplete` | Autocomplete input | `{ "type": "autocomplete", "name": "item" }` | Dynamic suggestions |

## Device Filter Syntax

```json
{
  "type": "device",
  "filter": "capabilities=onoff"
}
```

Multiple capabilities:
```json
{
  "filter": "capabilities=onoff+dim"
}
```

Driver filter:
```json
{
  "filter": "driver_id=switch_1gang"
}
```

## Range Args with Metadata

```json
{
  "type": "range",
  "name": "temperature",
  "min": 15,
  "max": 30,
  "step": 0.5,
  "label": "°C",
  "labelDecimals": 1,
  "title": { "en": "Target Temperature" }
}
```

## Dropdown with Translations

```json
{
  "type": "dropdown",
  "name": "mode",
  "values": [
    { "id": "auto", "label": { "en": "Auto", "fr": "Automatique" } },
    { "id": "heat", "label": { "en": "Heat", "fr": "Chauffage" } },
    { "id": "cool", "label": { "en": "Cool", "fr": "Refroidissement" } }
  ],
  "title": { "en": "Mode" }
}
```

## Best Practices

1. **Use `range` for numeric values** with known bounds (brightness 0-100, temperature 15-30)
2. **Use `dropdown` for fixed options** (modes, states, presets)
3. **Use `device` with filters** for cross-device actions
4. **Always provide translations** for `title` in en + fr minimum
5. **Use `label` on range** to show units (°C, %, W, lux)
6. **Use `labelDecimals`** to control precision display

## Common Patterns

### Temperature Threshold
```json
{
  "id": "trigger_temperature_above",
  "title": { "en": "Temperature above" },
  "args": [
    { "name": "threshold", "type": "range", "min": -20, "max": 60, "step": 0.5, "label": "°C" }
  ],
  "tokens": [
    { "name": "temperature", "type": "number", "title": { "en": "Current temperature" } }
  ]
}
```

### Brightness Control
```json
{
  "id": "action_set_brightness",
  "title": { "en": "Set brightness" },
  "args": [
    { "name": "brightness", "type": "range", "min": 0, "max": 100, "step": 1, "label": "%" }
  ]
}
```

### Mode Selection
```json
{
  "id": "action_set_mode",
  "title": { "en": "Set mode" },
  "args": [
    {
      "name": "mode",
      "type": "dropdown",
      "values": [
        { "id": "auto", "label": { "en": "Auto" } },
        { "id": "manual", "label": { "en": "Manual" } },
        { "id": "eco", "label": { "en": "Eco" } }
      ]
    }
  ]
}
```
