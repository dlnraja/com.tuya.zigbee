TECHNICAL NOTES
===============
SDK3 Requirement: Always use CLUSTER constants from 'zigbee-clusters':
- import { CLUSTER } from 'zigbee-clusters'
- Use CLUSTER.POWER_CONFIGURATION instead of 1
- Use CLUSTER.TEMPERATURE_MEASUREMENT instead of 1026
- Never register same capability twice

This is a CRITICAL fix that affects ALL battery-powered Zigbee devices.
Future drivers MUST follow this pattern to avoid runtime errors.
